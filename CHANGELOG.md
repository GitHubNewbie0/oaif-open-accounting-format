# Changelog

All notable changes to OAIF will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2] - 2026-04-20

### Added
- Standard payment terms seed data (7 common terms)

## [1.1] - 2026-01-12

### Added
- Attachment support for document storage
- Full platform mappings for 11 accounting systems

## [1.0] - 2026-01-01

### Added
- Initial OAIF specification
- Complete SQL schema with standard types
- 24 account types
- 125+ transaction types
- 13 item types
- Multi-currency support
- Investment/securities tracking
- Extension mechanism for custom fields
- QuickBooks to OAIF converter (qb2oaif)
